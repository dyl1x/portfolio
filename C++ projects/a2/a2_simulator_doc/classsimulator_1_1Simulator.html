Parameters

Returns
bool - Will return false if control can not be executed based on either of the conditions
(i) calculated gforce is more than G_MAX
(ii) the linear velocity is less than V_TERM
(iii) the linear velocity is more than V_MAX
(iv) control is supplied too frequently (more frequently than 25 ms apart)
(v) linear OR angular velocity are not numbers (nan)

◆ rangeBearingToBogiesFromFriendly()

std::vector< RangeBearingStamped > Simulator::rangeBearingToBogiesFromFriendly ( void  )

Returns the range from the friendly aircraft to the bogie.

Returns
RangeStamped - The range and bearing from friendly to bogies (metres) (rad), with a timestamp (milliseconds).
Note
This function is blocking at (FRIENDLY_REF_RATE), to enforce the refresh rate of the friendly aircraft.
The timestamp uses the simulation epoch timer as a reference.
Data in vectoris organised such that bogies with closest range are reported first an furthest last
